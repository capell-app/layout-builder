## What it does for you

Layout Builder is how you build and arrange the content on a page. You add widgets (blocks like text, images, buttons, and accordions), drag them into the order you want, and edit each one in place. You can save a block you have arranged as a reusable section and drop it onto other pages.

## Your screens

- **The page editor**: where you add widgets to a page and arrange them.
- **The widget panel**: the list of widget types you can drag onto the page.
- **Widgets**: the admin list of reusable widgets you have created.

## What you can do

- Add a widget to a page and position it by dragging.
- Edit a widget's text, image, and colours.
- Upload an image or file for a widget.
- Reorder widgets on the page.
- Save an arranged block as a reusable section.

## Where to find it

Open any page in the admin and edit its layout. The widget panel sits beside the page you are editing.

## Good to know

- Drag a widget from the panel onto your page, then click it to edit its text and image.
- Use **Save as section** to reuse a block on other pages.
- Changes to the page show on the live site once you save (and once any page cache clears).
